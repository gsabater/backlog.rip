**Public lists are here!**

More than half a year since the latest big update, finally I'm ready to release the public lists feature.
This feature was the last step to achieve feature parity with the old website (steam-backlog.com) and the end of the "beta" tag is getting closer.

Public lists are the same lists but now can be set as public and be viewed by anyone. And with it comes a new page to explore some of them. In addition to that, a new layout to edit lists is now available, making it easier to manage your lists and their contents.

Other updates include a new hompage, overall design improvements to the user interface, faster loading times, and various bug fixes. And finally, to add a a more technical note, the application has been deployed to Cloudflare Pages, which should improve performance and keep with site traffic.

## New features
- Lists can now be set as public and viewed by anyone
- Lists can now be deleted
- New page to explore public lists
- Deployment of the application to Cloudflare Pages

### Improvements
- A new homepage with a simpler animation to make the experience more enjoyable
- Improvements in the integrations, lists, states and cloud backup pages
- Old Icons library have been replaced with @nuxt/icons, which has more icons and is much faster to load
- Loading of content from the data sources now has more control and more consistent

### Fixes
- Fixed an issue with expired JWT tokens not being refreshed
- Server side rendering for games has been fixed, improving loading times and SEO
- Fixed a rare issue of race conditions when adding multiple games to a list
